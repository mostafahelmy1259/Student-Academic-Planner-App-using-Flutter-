# Student Academic Planner - Presentation Script

## Introduction

Our project is called Student Academic Planner. It is a mobile application built with Flutter and Firebase. The main goal is to help students organize their academic tasks, subjects, exams, reminders, and progress.

## Problem

Students usually have many assignments, exams, and deadlines. It can be difficult to remember everything or know what still needs to be done.

## Solution

Our app gives students one place where they can add tasks, manage subjects, schedule exams, set reminders, and see statistics about their progress.

## Technologies

We used Flutter and Dart for the mobile application. Firebase Authentication is used for login and registration. Cloud Firestore is used as the online database. Local notifications are used for reminders.

## Main features

- User registration and login
- Subject management
- Task management
- Exam schedule
- Reminders
- Statistics dashboard
- Firebase database security

## Database

The data is stored in Firestore. Each user has their own document under the users collection. Inside each user document, there are three subcollections: subjects, tasks, and exams.

## Conclusion

The app is useful, easy to use, and suitable for students. It demonstrates authentication, database CRUD operations, filtering, statistics, and notifications.
